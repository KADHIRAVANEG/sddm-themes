Ten self-contained SDDM greeter themes. Drop into /usr/share/sddm/themes/, pick your aesthetic, lock in.

$ sudo ./install.sh marvel
marvel Cinematic title card

all themes

three commands

git clone https://github.com/KADHIRAVANEG/sddm-themes
cd sddm-themes
sudo ./install.sh <theme-id>

Test without rebooting: sddm-greeter --test-mode --theme /usr/share/sddm/themes/<theme-id>

Requirements: SDDM ≥ 0.19 · Qt5 (qt5-quickcontrols2, qt5-graphicaleffects)
